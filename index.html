About

Basic Boilerplate is a bare-bones environment built around modern front end tools for automating basic front end development.

What the hell is Gulp, BrowserSync, Bower jargon?

Okay okay! Having a design focused background, I can relate to the confusion that comes with hearing this "jargon" for the first few times.

But, if you are wondering why go through all the fuss to get the same results as you would without using these tools; here are a tiny bit of reasons that define the purpose on surface!

  • Typically you download a whole bunch of jQuery or CSS libraries to support your creative ideas in the web. Then every time you would load them in your html files. Save yourself the trouble by just typing 3 words in that command line you despise.

  • You must be refreshing your page after every CSS style change! No? Ow so you use that browser plugin! But does it reload the same website on that smartphone of yours? Or that tablet you carry around? If that's a no then here you go. Typically you would update it on the hosting server of yours to view on other devices or better yet use some emulators in browser. But BrowserSync automagically sets up your local folder on your network to be viewed live from any home device, updating it on every single change.
